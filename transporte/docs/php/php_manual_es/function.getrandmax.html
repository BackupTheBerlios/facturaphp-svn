getrandmax

(PHP 3, PHP 4 , PHP 5)

getrandmax -- Muestra el mayor valor aleatorio posible

Descripción

int getrandmax ( void )

Devuelve el valor máximo que puede devolver una llamada a rand().

Vea también rand(), srand() mt_rand(), mt_srand() y mt_getrandmax().
