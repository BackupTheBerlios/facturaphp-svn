isset

(PHP 3, PHP 4, PHP 5 )

isset -- Determina si una varible está definida

Descripción

int isset ( mixed var)

Devuelve TRUE si var existe; y FALSE en otro caso.

Si una variable ha sido destruida con unset(), ya no estará definida (no será isset()). 

$a = "test";
echo isset($a); // true
unset($a);
echo isset($a); // false

Ver también empty() y unset().
