getcwd

(PHP 4 , PHP 5)

getcwd -- Obtiene el directorio de trabajo actual

Descripción

string getcwd ( void )

Devuelve el directorio de trabajo actual.

Ejemplo 1. Ejemplo de getcwd()

<?php

// directorio actual
echo getcwd() . "\n";

chdir('cvs');

// directorio actual
echo getcwd() . "\n";

?>

Este ejemplo producirá una salida similar a: 

/home/didou
/home/didou/cvs

Vea también chdir().
