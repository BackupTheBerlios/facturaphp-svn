is_nan

(PHP 4 >= 4.2.0, PHP 5)

is_nan -- Encuentra si un valor no es un número

Descripción

bool is_nan ( float val)

Devuelve TRUE si val no es un número ('not a number'), como el resultado de acos(1.01).

Vea también is_finite() y is_infinite().
