dio_write

(PHP 4 >= 4.2.0, PHP 5)

dio_write --  Escribe datos en el archivo cuyo descriptor es fd

Descripción

int dio_write ( resource fd, string data [, int len])

La función dio_write() escribe hasta len bytes de los datos indicados por el parámetro data en el archivo cuyo descriptor es fd. Si no se especifica el valor de len, la función dio_write() escribe todos los datos del parámetro data en el archivo indicado. La función dio_write() devuelve el numero de bytes escritos en el archivo cuyo descriptor es fd.
