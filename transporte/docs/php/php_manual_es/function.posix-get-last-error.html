posix_get_last_error

(PHP 4 >= 4.2.0, PHP 5)

posix_get_last_error --  Recuperar el número de error establecido por la última función posix que ha fallado.

Descripción

int posix_get_last_error ( void )

Devuelve el errno (número de error) establecido por la última función posix que falló. Si no existe error, el valor 0 es devuelto. Si desea contar con el mensaje de error del sistema asociado con el errno, utilice posix_strerror().

Vea también posix_strerror().
