mysqli_param_count -- Alias para mysqli_stmt_param_count()

Descripción

Esta función es un alias de mysqli_stmt_param_count(). Para una descipción detallada vea mysqli_stmt_param_count().

Nota: mysqli_param_count() es obsoleta y será removida.
