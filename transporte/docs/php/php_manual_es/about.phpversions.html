Versiones de PHP documentadas en este manual

Esta documentación contiene informacion sobre PHP 4, con algunas notas adicionales sobre migración y compatibilidad con PHP 3. Comportamientos, parámetros, valores retornados y otros cambios entre diferentes versiones de PHP se encuentran documentados en notas en este manual.

Podeis encontrar trozos de documentación para la versión CVS de PHP, esto significa la última versión de desarrollo disponible por el momento. Si no eres un desarrollador de PHP ó no utilizas la última versión de PHP en el CVS, no tendras disponibles las características marcadas con "disponible en el CVS". De todas maneras, estas características estarán probablemente disponibles en la proxima versión estable de PHP. Si deseais bajaros la versión del CVS, consultar la página sobre acceso anónimo al CVS.

También podeis encontrar documentación sobre una versión de PHP que no esté publicada todavia (p.ej PHP 5.0.0 cuando la última versión estable es 4.3.0). La mayoria de las veces no es un fallo. La explicación es añadida para características que se sabe estarán disponibles en una versión futura de PHP.
