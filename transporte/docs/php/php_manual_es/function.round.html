round

(PHP 3, PHP 4 , PHP 5)

round -- Redondea un float.

Descripción

double round ( double val)

Devuelve el valor redondeado de val. 

$foo = round( 3.4 );   // $foo == 3.0
$foo = round( 3.5 );   // $foo == 4.0
$foo = round( 3.6 );   // $foo == 4.0

Vea también ceil() y floor().
