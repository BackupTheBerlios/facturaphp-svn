imagesetthickness

(PHP 4 >= 4.0.6, PHP 5)

imagesetthickness -- Set the thickness for line drawing

Description

bool imagesetthickness ( resource image, int thickness)

imagesetthickness() sets the thickness of the lines drawn when drawing rectangles, polygons, ellipses etc. etc. to thickness pixels. Devuelve TRUE si todo se llevó a cabo correctamente, FALSE en caso de fallo.

Nota: This function was added in PHP 4.0.6 and requires GD 2.0.1 or later
