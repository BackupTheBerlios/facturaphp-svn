mysqli_stmt-init

(PHP 5)

mysqli_stmt-init

(no version information, might be only in CVS)

mysqli->stmt->init --  Inicializa una sentencia y regresa un objeto para ser usado con mysqli_stmt_prepare

Descripción

Estilo por procedimientos :

objeto mysqli_stmt_init ( objeto identificador_de_enlace)

Estilo orientado a objetos (propiedad):

class mysqli {

objeto stmt_init ( void )

}

Asigna e inicializa una objeto de sentencia conveniente para mysqli_stmt_prepare().

Nota: Cualquier llamada subsecuente a cualquier función mysqli_stmt fallará hasta que sea llamada mysqli_stmt_prepare().

Return values

Regresa un objeto.
