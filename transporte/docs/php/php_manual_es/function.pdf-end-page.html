PDF_end_page

(PHP 3>= 3.0.6, PHP 4 , PHP 5)

PDF_end_page -- Ends a page

Description

void pdf_end_page ( int pdf document)

The PDF_end_page() function ends a page. Once a page is ended it cannot be modified anymore.

See also PDF_begin_page().
