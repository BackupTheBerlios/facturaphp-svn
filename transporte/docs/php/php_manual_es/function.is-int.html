is_int

(PHP 3, PHP 4 , PHP 5)

is_int -- Averigua si una variable es un valor entero.

Descripción

int is_int ( mixed var)

Esta función es un alias de is_long().

Ver también is_double(), is_float(), is_integer(), is_string(), is_real(), is_object(), is_array(), y is_long().
