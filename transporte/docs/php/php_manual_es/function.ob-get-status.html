ob_get_status

(PHP 4 >= 4.2.0, PHP 5)

ob_get_status --  Obtener el status de los búferes de salida

Descripción

array ob_get_status ( [bool status_completo])

Aviso

Esta función es EXPERIMENTAL. Esto significa que el comportamiento de esta función, el nombre de esta función y en definitiva TODO lo documentado sobre esta función, puede cambiar en una futura version de PHP SIN AVISO. La advertencia queda hecha, y utilizar esta extensión queda bajo su propia responsabilidad.

Esta función devolverá el status actual de los búferes de salida. Devuelve una matriz que contiene el estátus de búfer, o FALSE en caso de fallo.

Vea también ob_get_level().
