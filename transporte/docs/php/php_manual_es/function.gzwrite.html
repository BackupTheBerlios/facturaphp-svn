gzwrite

(PHP 3, PHP 4 , PHP 5)

gzwrite -- Escritura en un archivo gz, segura con material binario

Descripción

int gzwrite ( resource az, string cadena [, int longitud])

gzwrite() escribe el contenido de cadena en la secuencia de archivo gz apuntada por az. Si el argumento longitud es definido, la escritura terminará después de que longitud bytes (sin comprimir) sean escritos o se alcance el final de cadena, lo que ocurra primero.

gzwrite() devuelve el número de bytes (sin comprimir) escritos a la secuencia de archivo gz apuntada por az.

Note que si se define el argumento longitud, entonces la opción de configuración magic_quotes_runtime será ignorada y no se removerán caracteres barra de cadena.

Vea también gzread(), gzopen(), y gzputs().
