getmyuid

(PHP 3, PHP 4 , PHP 5)

getmyuid -- Obtiene el UID del propietario del script PHP.

Descripción

int getmyuid ( void )

Devuelve el ID de usuario del script actual, o FALSE en caso de error.

Véase también getmypid(), get_current_user(), getmyinode(), y getlastmod().
