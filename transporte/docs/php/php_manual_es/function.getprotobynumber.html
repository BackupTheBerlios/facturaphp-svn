getprotobynumber

(PHP 4 , PHP 5)

getprotobynumber --  Obtener el nombre de protocolo asociado con un número de protocolo

Descripción

string getprotobynumber ( int numero)

getprotobynumber() devuelve el nombre de protocolo asociado con el numero de procotolo, como lo indica /etc/protocols.

Vea también: getprotobyname().
