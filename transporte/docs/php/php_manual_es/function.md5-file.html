md5_file

(PHP 4 >= 4.2.0, PHP 5)

md5_file -- Calcula el resumen criptográfico md5 de un nombre de archivo dado

Descripción

string md5_file ( string nombre_archivo [, bool salida_primitiva])

Calcula el resumen criptográfico MD5 del nombre_archivo especificado, usando el Algoritmo de Resumen de Mensajes MD5 de RSA Data Security, Inc., y devuelve ese resumen. El resumen criptográfico es un número hexadecimal de 32 caracteres. Si el valor opcional salida_primitiva es TRUE, entonces el resumen md5 es retornado en cambio en formato binario primitivo con una longitud de 16.

Nota: El parámetro opcional salida_primitiva fue añadido en PHP 5.0.0 y tiene un valor predeterminado de FALSE

Esta función tiene el mismo propósito que la utilidad de línea de comandos md5sum.

Vea también md5(), crc32(), y sha1_file().
