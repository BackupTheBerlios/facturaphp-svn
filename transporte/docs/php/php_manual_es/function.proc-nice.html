proc_nice

(PHP 5)

proc_nice --  Modificar la prioridad del proceso actual

Descripción

bool proc_nice ( int prioridad)

proc_nice() modifica la prioridad del proceso actual. Si un error ocurre, como que el usuario carezca de permisos para modificar la prioridad, un error de nivel E_WARNING es generado, y se devuelve FALSE. De otro modo, se devuelve TRUE.

Nota: proc_nice() existirá únicamente si su sistema tiene soporte para 'nice'. 'nice' está definido de acuerdo a los estándares: SVr4, SVID EXT, AT&T, X/OPEN, BSD 4.3. Esto quiere decir que proc_nice() no está disponible en windows.

La función proc_nice() no se encuentra relacionada con proc_open() ni sus funciones asociadas en ninguna forma.
