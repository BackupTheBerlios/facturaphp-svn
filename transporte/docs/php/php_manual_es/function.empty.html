empty

(PHP 3, PHP 4, PHP 5 )

empty -- Determina si una variable está definida

Descripción

int empty ( mixed var)

Devuelve FALSE si var está definida y tiene un valor no-vacío o distinto de cero; en otro caso devuelve TRUE.

Ver también isset() y unset().
