is_real

(PHP 3, PHP 4 , PHP 5)

is_real -- Averigua si una varible es un número real.

Descripción

int is_real ( mixed var)

Esta función es un alias de is_double().

Ver también is_long(), is_int(), is_integer(), is_float(), is_double(), is_object(), is_string(), y is_array().
