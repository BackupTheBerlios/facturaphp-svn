gettype

(PHP 3, PHP 4 , PHP 5)

gettype -- Obtiene el tipo de una variable.

Descripción

string gettype ( mixed var)

Devuelve el tipo de la variable PHP var.

Los valores posibles de la cadena devuelta son:

  • "integer"

  • "double"

  • "string"

  • "array"

  • "object"

  • "unknown type"

Ver también settype().
