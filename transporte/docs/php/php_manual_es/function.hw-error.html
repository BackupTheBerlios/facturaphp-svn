hw_Error

(PHP 3>= 3.0.3, PHP 4 )

hw_Error -- número de error

Descripción

int hw_error ( int conexión)

Devuelve el último número de error. Si el valor devuelto es 0 no ha habido errores. El error está relacionado con el último comando.
