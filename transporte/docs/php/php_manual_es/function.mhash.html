(PHP 3>= 3.0.9, PHP 4 , PHP 5)

mhash -- Calcular el hash

Descripcion

string mhash ( int hash, string data)

mhash() aplica una funcion hash especificada por hash a data y devuelve el valor hash resultante (tambien llamdo digest).
