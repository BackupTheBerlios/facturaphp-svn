imap_expunge

(PHP 3, PHP 4 , PHP 5)

imap_expunge -- Elimina todos los mensajes marcados como borrados

Descripción

int imap_expunge ( int imap_stream)

imap_expunge() elimina todos los mensajes marcados por la función imap_delete().

Devuleve TRUE.
