ImageFontWidth

(PHP 3, PHP 4 , PHP 5)

ImageFontWidth -- Devuelve la anchura de una fuente

Descripción

int imagefontwidth ( int font)

Devuelve la anchura en pixels de un carácter en un fuente específica.

Vea también imagefontheight() y imageloadfont().
