hw_Root

(PHP 3>= 3.0.3, PHP 4 )

hw_Root -- ID del objeto raíz

Descripción

int hw_root ( )

Devuelve la ID de objeto de la colección hiper-raíz. Actualmente siempre vale 0. La colección hija de la hiper-raíz es la colección raíz del servidor al que se ha conectado.
