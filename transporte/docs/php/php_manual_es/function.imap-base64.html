imap_base64

(PHP 3, PHP 4 , PHP 5)

imap_base64 -- Decodifica texto codificado en BASE64

Descripción

string imap_base64 ( string text)

imap_base64() decodifica texto codificado en BASE-64. El mensaje decodificado es devuelto como una cadena.
