chop

(PHP 3, PHP 4 , PHP 5)

chop -- Elimina espacios sobrantes al final

Descripción

string chop ( string cad)

Devuelve la cadena argumento sin los espacios sobrantes, incluyendo los saltos de línea.

Ejemplo 1. Ejemplo de chop()

$recortada = chop ($linea);

Vea también trim().
