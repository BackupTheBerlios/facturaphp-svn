unset

(PHP 3, PHP 4, PHP 5 )

unset -- Destruye una variable dada

Descripción

int unset ( mixed var)

unset() destruye la variable especificada y devuelve TRUE.

Ejemplo 1. Ejemplo de unset()

unset( $foo );
unset( $bar['quux'] );

Ver también isset() y empty().
