tanh

(PHP 4 >= 4.1.0, PHP 5)

tanh -- Tangente hiperbólica

Descripción

float tanh ( float arg)

Devuelve la tangente hiporbólica de arg, definida como sinh(arg)/cosh(arg).

Vea también: tan(), atanh(), sin(), y cos().
