getmygid

(PHP 4 >= 4.1.0, PHP 5)

getmygid -- Obtener el GID del dueño del script PHP

Descripción

int getmygid ( void )

Devuelve el ID de grupo del script actual, o FALSE en caso de fallo.

Vea también getmyuid(), getmypid(), get_current_user(), getmyinode(), y getlastmod().
