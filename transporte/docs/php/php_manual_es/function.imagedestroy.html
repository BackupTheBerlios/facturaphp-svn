ImageDestroy

(PHP 3, PHP 4 , PHP 5)

ImageDestroy -- Destruye una imagen

Descripción

int imagedestroy ( int im)

ImageDestroy libera la memoria asociada a la imagen im. im es la imagen devuelta por la función imagecreate().
