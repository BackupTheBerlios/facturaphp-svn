(PHP 3, PHP 4 , PHP 5)

dbase_pack -- "empaqueta" un fichero dBase

Descripción

bool dbase_pack ( int dbase_identifier)

Empaqueta el fichero especificado, borrando definitivamente todos los registros marcados con la función dbase_delete_record().
