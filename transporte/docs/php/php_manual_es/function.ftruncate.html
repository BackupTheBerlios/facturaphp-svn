ftruncate

(PHP 4 , PHP 5)

ftruncate -- Trunca un archivo a la longitud dada

Descripción

bool ftruncate ( resource gestor, int tamanyo)

Toma el apuntador de archivo, gestor, y trunca el archivo a la longitud tamanyo. Devuelve TRUE si todo se llevó a cabo correctamente, FALSE en caso de fallo.

Nota: En versiones anteriores a PHP 4.3.3, ftruncate() devuelve un valor integer de 1 cuando tiene éxito, en lugar del valor boolean TRUE.

Vea también fopen() y fseek().
