ob_get_clean

(PHP 4 >= 4.3.0, PHP 5)

ob_get_clean --  Obtener los contenidos del búfer actual y eliminar el búfer de salida actual

Descripción

string ob_get_clean ( void )

Esta función devolverá los contenidos del búfer de salida y finalizará el control de salida mediante búferes. Si el control de salida con búferes no está activo, entonces se devuelve FALSE. ob_get_clean() básicamente ejecuta tanto ob_get_contents() como ob_end_clean().

Ejemplo 1. Un ejemplo simple de ob_get_clean()

<?php

ob_start();

echo "Hola Mundo";

$salida = ob_get_clean();
$salida = strtolower($salida);

var_dump($salida);
?>

Nuestro ejemplo producirá la salida: 

string(10) "hola mundo"

Vea también ob_start() y ob_get_contents().
