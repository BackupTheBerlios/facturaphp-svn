nl2br

(PHP 3, PHP 4 , PHP 5)

nl2br -- Convierte nuevas líneas a saltos de línea HTML

Descripción

string nl2br ( string cadena)

Devuelve la cadena con '<BR>' insertados antes de cada nueva línea.

Vea también htmlspecialchars(), htmlentities() y wordwrap().
