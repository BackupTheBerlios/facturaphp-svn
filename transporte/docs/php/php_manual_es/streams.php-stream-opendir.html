php_stream_opendir -- Abre un directorio para la enumeración de archivos

Descripción

php_stream * php_stream_opendir ( char * ruta, php_stream_context * contexto)

php_stream_opendir() devuelve una secuencia que puede ser usada para listar los archivos contenidos en el directorio especificado por ruta. Esta función es equivalente en la práctica a la función POSIX opendir(). Aunque ésta función devuelve un objeto php_stream, no se recomienda usar las funciones de la API común sobre éstas secuencias.
