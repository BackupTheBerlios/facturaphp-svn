abs

(PHP 3, PHP 4 , PHP 5)

abs -- Valor absoluto

Descripción

mixed abs ( mixed number)

Devuelve el valor absoluto de un número. Si el número del argumento es decimal, el tipo de retorno también es decimal, de otra manera devuelve un entero.
