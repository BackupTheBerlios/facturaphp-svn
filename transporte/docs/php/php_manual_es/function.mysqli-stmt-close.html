mysqli_stmt_close

(PHP 5)

mysqli_stmt_close

(no version information, might be only in CVS)

mysqli_stmt->close -- Cierra la sentencia preparada

Descripción

Estilo por procedimientos:

bool mysqli_stmt_close ( objeto stmt)

Estilo orientado a objetos (método):

class mysqli_stmt {

bool mysqli_stmt->close ( void )

}

Cierra una sentencia preparada. mysql_stmt_close() también desaloja el manejador de sentencias apuntado por stmt. Si la sentencia actual tiene resultados pendientes o no leídos, esta función los cancela de tal manera que la siguiente consulta pueda ser ejecutada.

Valores regresados

Devuelve TRUE si todo se llevó a cabo correctamente, FALSE en caso de fallo.
